This program applies multinomial logit model to predict the probability of the type of vehicle that is owned. 

The alternatives are: 
Body type: car, van, SUV, pickup, motorcycle
fuel type: gas, diesel, hybrid, electric
age: up to 20 years

The inputs are: 
vehicle cost and EPA vehicle data
NHTS 2017 data
coefficients and logit model
CHOICE 4: listed as body type, fuel type and age


The ouputs are:
Table containing household ID and the 500 choices from body type, age and fuel type
probability of picking each choice for each household, should add up to 1
Utility equation is used to calculate each choice, and probability is gotten from exponent of utility / sum of exponents of utilities
